# Global search across cases, runs, and suites

## Problem
No cross-entity search. Finding a case referenced in a run requires switching tabs and searching separately.

## What to build
- Search bar in NavBar (cmd+K shortcut)
- Client-side fuzzy search across already-fetched cases, runs, suites using Fuse.js
- Results grouped by type with direct navigation
- Falls back to per-tab search when data not yet loaded

## Files
- `web/src/components/NavBar.tsx`
- New `GlobalSearch.tsx` component
