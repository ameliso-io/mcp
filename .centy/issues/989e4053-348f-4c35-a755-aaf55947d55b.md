# Sync suite YAML changes via GitHub webhook

## Problem
`webhook.rs` only processes `.ameliso/cases/*.md` files on push. Suite YAML edits committed directly to git don't update the DB — suite sync is manual-only via `SyncRepository`.

## What to build
- Extend `is_case_file` check to also detect `.ameliso/suites/*.yaml`
- On upsert: parse suite YAML and call `upsert_suite` equivalent
- On remove: delete suite from DB

## Files
- `server/src/webhook.rs` — `is_case_file`, `collect_case_changes`, `github_push`
- `server/src/repo/` — add `upsert_suite` if not present
