# Export run report as CSV from RunsTab

## Why

QA leads need to share test results with stakeholders who don't have Ameliso access. No export means manual copy-paste from the UI into spreadsheets.

## Done when

- "Export CSV" button on finalized run detail view
- CSV columns: `case_path, status, notes, tester, recorded_at`
- Filename: `{run_id}-results.csv`
- Export is client-side (no new RPC) — builds CSV from already-fetched result data
- Button disabled / hidden for in-progress runs
