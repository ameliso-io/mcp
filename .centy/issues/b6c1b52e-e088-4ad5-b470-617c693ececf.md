# Dark mode support

## Problem
No theme support. CSS modules use hardcoded color values. Night testing sessions (common) require full-brightness white UI.

## What to build
- CSS custom properties (variables) for all colors in `globals.css`
- `@media (prefers-color-scheme: dark)` overrides
- Manual toggle in NavBar with `localStorage` persistence

## Files
- `web/app/globals.css`
- `web/src/components/NavBar.tsx` — theme toggle
- All `*.module.css` files — replace hardcoded colors with CSS vars
