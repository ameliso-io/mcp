# Add tsc --noEmit to web pre-push hook

## Problem
The web pre-push hook runs build and tests but not a standalone tsc --noEmit check. TypeScript errors in files not covered by tests can slip through.

## Fix
Add pnpm test:typecheck (tsc --project tsconfig.test.json --noEmit) to the web pre-push hook so type errors in test files are caught before push.
