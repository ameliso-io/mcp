# Virtualize long case and run lists for performance

## Problem
`RunsTab.tsx` (1274 lines) and `CasesTab.tsx` (1018 lines) render full lists. Large repos with many cases/runs will have UI perf issues.

## What to build
- Implement virtual scrolling for case list in CasesTab
- Implement virtual scrolling for run list in RunsTab
- Use `@tanstack/react-virtual` or similar

## Files
- `web/src/components/CasesTab.tsx`
- `web/src/components/RunsTab.tsx`
