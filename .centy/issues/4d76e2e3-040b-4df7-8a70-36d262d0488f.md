# Set up Vitest testing infrastructure for web crate

Install and configure Vitest + React Testing Library in `web/`.

## Tasks
- Add `vitest`, `@testing-library/react`, `@testing-library/user-event`, `@testing-library/jest-dom` as dev dependencies
- Configure `vite.config.ts` with test block (environment: jsdom)
- Add `test` script to `web/package.json`
- Add `test` target to root `Makefile`
- Set up mock for gRPC-web client (`web/src/client.ts`) so components can be tested without a live server

## Notes
- Use `pnpm` (project uses pnpm workspaces)
- Mock strategy: `vi.mock` on `../client` in each test file, or a shared `__mocks__/client.ts`
