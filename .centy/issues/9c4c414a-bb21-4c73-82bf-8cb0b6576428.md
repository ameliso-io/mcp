# Auto-load pending cases after run creation

## Problem
After creating a run, user must manually navigate to the run to see pending cases. Extra click, bad for agent workflows.

## What to build
- After successful `CreateRun`, immediately call `GetPendingCases` and display results
- Auto-select the new run in the runs list
- Optionally scroll to the run's case list
