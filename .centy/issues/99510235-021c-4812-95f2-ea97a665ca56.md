# Shareable HTML run report export

## Problem
No way to share run results with stakeholders who don't have Ameliso access. Email/archive requires manual copy-paste.

## What to build
- 'Export report' button on completed run
- Generate self-contained HTML: case list with status badges, notes, tester, environment, commit_sha, pass/fail summary
- Client-side generation from already-fetched `GetRun` response — no new RPC
- Download as `run-{run_id}.html`
