# Add Husky and implement pre-commit/pre-push hooks via Makefile

## Summary

Add Husky to the root of the repo and wire up pre-commit and pre-push Git hooks that delegate to Makefile targets.

## Tasks

- Install Husky at repo root (`pnpm dlx husky init` or equivalent)
- Add `.husky/pre-commit` hook that runs `make pre-commit`
- Add `.husky/pre-push` hook that runs `make pre-push`
- Add `pre-commit` target to Makefile (e.g. lint, format check, unit tests)
- Add `pre-push` target to Makefile (e.g. full test suite, build verification)
- Ensure hooks are executable and committed to the repo
- Document hook targets in README or Makefile comments

## Acceptance Criteria

- `git commit` triggers Makefile `pre-commit` target
- `git push` triggers Makefile `pre-push` target
- Hooks fail the operation on non-zero exit code
- Works for all contributors after `pnpm install` (husky prepare script)
