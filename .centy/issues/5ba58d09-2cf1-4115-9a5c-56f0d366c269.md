# Drag-to-reorder cases within a suite

## Problem
Suite case list has no ordering — cases appear in insertion order. Execution order matters (smoke tests first). No way to reorder without delete + re-add.

## What to build
- Drag-and-drop reorder in `SuiteExpandedPanel` case list
- On drop: call `UpdateSuite(replace_cases=true, cases=[...reordered])`
- Use `@dnd-kit/sortable` or similar
