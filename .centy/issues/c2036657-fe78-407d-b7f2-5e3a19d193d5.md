# Tag management UI (global tag list, rename, merge)

## Problem
Tags are free-form strings with no governance. Typos (e.g. `authn` vs `auth`) silently create parallel tags. No way to see all tags, rename them, or merge duplicates.

## What to build
- Tag management panel (in CasesTab sidebar or Settings page)
- List all unique tags across the repo with case counts
- Rename tag: updates all cases with that tag via `BulkUpdateCases`
- Delete tag: removes from all cases

## Files
- `web/src/components/CasesTab.tsx` or new `TagManager.tsx`
- Backend: may need a `ListTags` RPC or derive from `ListCases` response
