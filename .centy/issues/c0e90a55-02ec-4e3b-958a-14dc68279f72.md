# Add .env.example with all required environment variables

## Problem
Required environment variables (DATABASE_URL, GITHUB_APP_ID, GITHUB_APP_PRIVATE_KEY, WEBHOOK_SECRET, etc.) are referenced in code but not documented in one place. New contributors get cryptic startup failures.

## Fix
Create .env.example at repo root listing every required and optional env var with description and example value. Add runtime validation in server/main.rs that names any missing required variable at startup.
