# Add bulk case operations to CasesTab (select, assign suite, delete)

## Why

Users managing large case libraries must act on cases one at a time. No way to assign 20 cases to a suite or delete a stale batch without 20 individual clicks.

## Done when

- Checkbox column added to cases table (select-all + per-row)
- Bulk action bar appears when ≥1 case selected: "Assign to suite", "Delete selected"
- Assign: shows suite picker dropdown → calls `BulkUpdateCases` with suite assignment
- Delete: confirmation dialog → calls `BulkDeleteCases`
- Selection cleared after action completes
- Works with existing tag/priority filters (only visible cases can be bulk-selected)
