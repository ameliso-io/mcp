# Testing Repo Structure

## Directory Layout

```
/
├── cases/
│   └── {category}/
│       └── {subcategory}/
│           └── ...
│               └── {slug}.md
├── suites/
│   └── {slug}.yaml
└── runs/
    └── {YYYY-MM-DD}-{slug}/
        ├── run.yaml
        └── results/
            └── {category}/
                └── {subcategory}/
                    └── ...
                        └── {case-slug}.md
```
